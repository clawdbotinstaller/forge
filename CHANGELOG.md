## [0.1.0] - 2026-02-25

### Added
- Initial release of FORGE master plugin
- 9-phase development workflow:
  - Brainstorm: Multi-agent exploration with Karpathy-guided debate
  - Research: BMAD-style parallel research subagents
  - Design: Stitch-powered UI generation with design system compliance
  - Plan: Surgical implementation planning with TDD
  - Build: Subagent-driven development with two-stage review
  - Test: WHOLE PHASE dedicated to comprehensive test suite creation
  - Validate: Evidence-based quality validation
  - Review: Parallel specialized code review
  - Learn: Continuous learning with CLAUDE.md auto-updates
- Smart orchestrator (`/forge:help`) for workflow guidance
- Quick mode (`/forge:quick`) for rapid development
- Configurable artifact levels (minimal/intelligent/maximal)
- Quality gates with evidence requirements
- Dynamic skill loading based on task type
- Progressive context building (BMAD pattern)
- Karpathy guidelines enforcement throughout
- Helper scripts for quality gates, artifact management, and learning extraction

### Components
- 19 skills (3 core + 10 workflow + 6 agents)
- 11 commands (9 phase + 2 orchestrator)
- 3 helper scripts
